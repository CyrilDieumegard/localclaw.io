What does beta lifetime access mean?
LocalClaw is available while it is still improving. The beta license gives you lifetime access to the macOS app for a one-time $49 purchase, with updates included as the product matures. Features and setup flows may change during beta.
What are the system requirements?
LocalClaw is a macOS app for macOS 13 Ventura or later. Apple Silicon is recommended, Intel Macs are supported where the local AI stack works. 8 GB RAM is the minimum; 16 GB or more is strongly recommended for practical local models.
Is the DMG notarized by Apple?
Yes. The current LocalClaw DMG is signed with a Developer ID certificate and notarized by Apple, so macOS Gatekeeper can verify it before launch. Notarization is Apple security scanning and ticketing; it is not App Store distribution or an Apple endorsement.
How does activation work?
After Stripe checkout, your license key is shown on the confirmation page. Download the installer, launch LocalClaw, enter your order email and license key, then activate. Activation uses a one-time internet check.
Is the free model recommender still free?
Yes. The web recommender, model catalogue, TTS catalogue and hardware guides stay free. The $49 purchase is only for the optional native macOS app.
What if the install or license key has a problem?
Email support is included for activation, missing keys and install guidance. Use the same email address used during checkout so the order can be verified quickly.
Does LocalClaw collect data?
The LocalClaw website recommender runs in your browser. The app is built around local OpenClaw workflows on your machine. Payment and license activation require Stripe and a license check, but the product is designed for local AI control, not cloud data collection.
