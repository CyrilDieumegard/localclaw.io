What changed

Live agent progress

Developer shows the current phase, elapsed time, selected model, timeout, and recent tool actions while OpenClaw works on the project.

Embedded responsive preview

Run the app inside LocalClaw, refresh after code changes, and switch between desktop and mobile widths without opening another browser window.

Workspace and Git visibility

Project, branch, changed-file count, and preview state stay visible. Files, Git, database, deploy tools, and logs remain one click away.

More reliable Developer sessions

Project isolation, fresh-context handling, scrolling, managed provider authentication, and automatic Gateway recovery make longer build sessions easier to follow and resume.

Existing customers

Open Updates in LocalClaw and install version 1.0.175. Your OpenClaw configuration, projects, channels, local models, and existing license remain in place.
