What was used

This demonstration uses Kimi K3 through OpenRouter. LocalClaw also supports local and OAuth model routes, but the model used for this recorded build was cloud-hosted.

  • InterfaceLocalClaw Developer with project chat, files, tools and embedded live preview.
  • Feedback loopPlayable preview used to test movement, combat, enemies and visual changes between iterations.
